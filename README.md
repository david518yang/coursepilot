### Getting Started

#### Requirements

- [Node.js]()
- [pnpm](https://pnpm.io/installation)

after cloning the repo, navigate into `/frontend` and run

```zsh
pnpm i && pnpm dev
```

to install coursepilot's dependencies and run the development server, hosted locally at `http://localhost:3000/`
